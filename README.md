# Zydoc - AI Healthcare 🏥
**Developed By: Shani Bokade**

Zydoc is a comprehensive AI-powered Hospital Management System designed to streamline medical operations and provide intelligent symptom analysis for patients.

## 🚀 Key Features
* **AI Symptom Analysis:** Integrated chatbot for preliminary medical guidance.
* **Doctor/Patient Dashboards:** Dedicated portals for managing appointments and prescriptions.
* **Automated Invoicing:** Generate and download PDF invoices upon patient discharge.
* **Real-time Management:** Admin controls for approving doctors and tracking hospital capacity.

## 🛠️ Technology Stack
* **Backend:** Python, Django
* **Database:** PostgreSQL / SQLite
* **Frontend:** HTML, CSS, Bootstrap
* **AI:** Natural Language Processing (NLP) for medical data analysis
